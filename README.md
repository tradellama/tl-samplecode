## Running the tests

Once you have Go installed, yo ucan run this at the root of the repository

```
$ go test -cover  ./...
```

And the result will be

```
ok      github.com/tradellama/tl-samplecode/movingaverage       0.001s  coverage: 100.0% of statements
```
